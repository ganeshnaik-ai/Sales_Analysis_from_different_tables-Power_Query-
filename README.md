# Sales Analysis from Different Data Sets (Power Query)

## Project Overview
This project demonstrates the use of **Power Query** in Excel to perform ETL (Extract, Transform, Load) processes on fragmented data. By connecting and merging three separate tables—**Customers**, **Products**, and **Orders**—this interactive dashboard provides actionable insights into sales trends, customer behavior, and operational efficiency.

## Features
* **Data Integration:** Merged multiple relational tables into a single data model using Power Query.
* **Interactive Slicers:** Filter the entire dashboard by Occasion, Product Category, or Time Period.
* **Automated Metrics:** Real-time calculation of KPIs like Average Delivery Time and Revenue per Customer.
* **Visual Trend Analysis:** Month-over-month sales and order volume tracking.

## Key Insights
* **Revenue Leader:** The "Colors" category drives the highest revenue.
* **Seasonality:** Significant spikes in sales are observed during February and August, correlating with major occasions.
* **Efficiency:** The average delivery time is maintained at ~5.5 days across 1,000 orders.

## Technologies Used
* **Microsoft Excel:** Data visualization and UI.
* **Power Query:** Data cleaning, merging, and transformation.
* **DAX/Excel Formulas:** For KPI calculations and custom metrics.

## How to Use
1. Clone this repository.
2. Open the `.xlsx` file.
3. If prompted, enable external data connections.
4. Use the slicers on the right side of the dashboard to filter the data by your preferred criteria.
